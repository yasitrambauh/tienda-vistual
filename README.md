# 🛒 Tienda Visual

Este es un proyecto de tienda virtual básica que integra una interfaz web con una base de datos SQL.

## 📁 Contenido del Repositorio

* **DecorAPP.html**: Interfaz principal de la aplicación desarrollada en HTML.
* **tiendavirtual.sql**: Script para la creación y configuración de la base de datos necesaria para el funcionamiento del sistema.
* **db_tiendavirtual.txt**: Documentación adicional o respaldo de la estructura de datos.

## 🚀 Cómo empezar

1.  **Base de Datos**: Importa el archivo `tiendavirtual.sql` en tu gestor de bases de datos (como MySQL o MariaDB).
2.  **Interfaz**: Abre el archivo `DecorAPP.html` en cualquier navegador moderno para visualizar la tienda.

## 🛠️ Tecnologías utilizadas

* HTML5 / CSS
* SQL (MySQL)
